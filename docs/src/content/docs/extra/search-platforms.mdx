Ryanlink supports a vast array of search platforms. When performing a search using `manager.search()` or `player.search()`, pass a URL or a query string prefixed with one of the following identifiers.

## NodeLink Professional Sources <Badge text="Requires NodeLink" variant="tip" />

NodeLink provides native, high-performance support for an extensive range of global and regional platforms.

| Category | Supported Platforms |
| :--- | :--- |
| **Streaming** | `spotify`, `applemusic`, `deezer`, `tidal`, `amazonmusic`, `qobuz`, `audiomack`, `pandora`, `lastfm`, `gaana`, `jiosaavn`, `anghami`, `audius`, `netease`, `vkmusic`, `yandexmusic`, `myfreemp3` |
| **Video & Music** | `youtube`, `youtube music`, `soundcloud`, `bandcamp`, `bilibili`, `nicovideo`, `vimeo`, `twitch`, `mixcloud`, `monochrome` |
| **Social & News** | `instagram`, `tiktok`, `twitter`, `tumblr`, `reddit`, `pinterest`, `bluesky`, `telegram`, `rss` |
| **Regional Content** | `kwai` (CN), `gaana` (IN), `jiosaavn` (IN), `netease` (CN), `vkmusic` (RU), `yandexmusic` (RU), `anghami` (MENA) |
| **TTS & Utilities** | `google-tts`, `pipertts`, `flowery`, `lazypytts`, `letrasmus`, `genius`, `shazam`, `songlink`, `eternalbox`, `http`, `local` |

## LavaSrc & LavaSearch Support <Badge text="Plugin Required" variant="caution" />

When using the LavaSrc plugin, Ryanlink enables advanced features like ISRC resolution, recommendations, and rich metadata for these platforms:

| Platform | Search Prefix | Features |
| :--- | :--- | :--- |
| **Spotify** | `spsearch:` | Albums, Playlists, Tracks, Recommendations |
| **Apple Music** | `amsearch:` | Albums, Playlists, Tracks |
| **Deezer** | `dzsearch:` | Albums, Playlists, Tracks, ISRC |
| **YouTube Music** | `ytmsearch:` | Video & Audio isolation |
| **Tidal** | `tdsearch:` | High-fidelity track resolution |
| **JioSaavn** | `jssearch:` | Regional playlists and charts |
| **Amazon Music** | `amzsearch:` | Full catalog search |
| **Instagram** | `igsearch:` | Audio from Reels and Posts |
| **Google TTS** | `google-tts:` | Cloud-powered text-to-speech |
| **Mixcloud** | `mixcloud:` | Long-form radio and DJ sets |

## Setting a Default Platform

```typescript
const manager = new RyanlinkManager({
  playerOptions: {
    defaultSearchPlatform: 'spsearch', // Spotify as default
  }
});
```

Any query that isn't a direct URL will automatically use this prefix.

## Source Validation

When `autoChecks.sourcesValidations` is `true` (default), Ryanlink validates that the node has the required source manager enabled before sending the request. Set it to `false` to skip validation:

```typescript
nodes: [{
  id: 'main',
  host: 'localhost',
  port: 2333,
  authorization: 'youshallnotpass',
  autoChecks: {
    sourcesValidations: false,
    pluginValidations: false,
  }
}]
```
