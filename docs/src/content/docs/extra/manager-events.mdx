The `RyanlinkManager` emits events throughout the lifecycle of nodes and players. You can listen to these events to update your UI or handle errors proactively.

## Node Events

| Event | Payload | Description |
| :--- | :--- | :--- |
| `nodeConnect` | `(node: Node)` | Emitted when a node successfully establishes a WebSocket connection. |
| `nodeDisconnect` | `(node: Node)` | Emitted when a node's WebSocket connection is severed. |
| `nodeReconnect` | `(node: Node)` | Emitted when a node is attempting to reconnect. |
| `nodeError` | `(node: Node, error: Error)` | Emitted when a node encounters a critical communication error. |
| `nodeRaw` | `(node: Node, payload: any)` | Emitted for every raw packet received from the Lavalink node. |

---

## Player Events

| Event | Payload | Description |
| :--- | :--- | :--- |
| `trackStart` | `(player: Player, track: Track)` | Emitted when a track begins playing. |
| `trackEnd` | `(player: Player, track: Track, reason: string)` | Emitted when a track finishes playing. |
| `trackStuck` | `(player: Player, track: Track, threshold: number)` | Emitted when a track gets stuck during playback. |
| `trackError` | `(player: Player, track: Track, error: any)` | Emitted when an error occurs during track playback. |
| `queueEnd` | `(player: Player)` | Emitted when the queue is empty and playback finishes. |
| `playerDestroy` | `(player: Player, reason: string)` | Emitted when a player is destroyed (Kicked/Closed). |

---

## Smart Engine Events <Badge text="New in 2.0" variant="tip" />

| Event | Payload | Description |
| :--- | :--- | :--- |
| `queueErrorReport` | `(p: Player, track: Track, error: Error)` | Emitted when an UnresolvedTrack fails all retry attempts. |
| `nodeFailover` | `(p: Player, oldNode: Node, newNode: Node)` | Emitted when an Atomic Migration is successfully completed. |

---

## Usage Example

```typescript
manager.on('trackStart', (player, track) => {
  console.log(`[${player.guildId}] Now playing: ${track.info.title}`);
});

manager.on('queueErrorReport', (player, track, error) => {
  console.error(`[${player.guildId}] Failed to resolve: ${track.info.title} after 3 retries.`);
});
```
