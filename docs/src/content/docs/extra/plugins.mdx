## SponsorBlock <Badge text="sponsorblock-plugin" variant="caution" />

Automatically skip sponsor segments, intros, outros, and more in YouTube videos.

SponsorBlock is **auto-configured** by Ryanlink on every player's first track start and on node reconnect. You do not need to call `setSponsorBlock` manually unless you want custom segments.

### Auto-Configuration

Ryanlink calls `player.setSponsorBlock(['sponsor', 'selfpromo'])` automatically when:
- A track starts playing for the first time on a player
- A node reconnects and existing players are restored
- A player migrates to a new node (`changeNode`)

Enable this behaviour by setting `enforceSponsorBlockRequestForEventEnablement: true` in your manager options (this is the default):

```typescript
const manager = new RyanlinkManager({
  playerOptions: {
    enforceSponsorBlockRequestForEventEnablement: true,
  }
});
```

### Manual Control

```typescript
// Override with custom segments
await player.setSponsorBlock(['sponsor', 'selfpromo', 'interaction', 'intro', 'outro', 'preview', 'music_offtopic', 'filler']);

// Get current active segments
const categories = await player.getSponsorBlock();

// Disable all segment skipping
await player.deleteSponsorBlock();
```

### Events

```typescript
manager.on('SegmentsLoaded', (player, track, payload) => {
  console.log(`Segments loaded: ${payload.segments.map(s => s.category).join(', ')}`);
});

manager.on('SegmentSkipped', (player, track, payload) => {
  console.log(`Skipped: ${payload.segment.category} (${payload.segment.start}–${payload.segment.end}ms)`);
});

manager.on('ChaptersLoaded', (player, track, payload) => {
  console.log(`${payload.chapters.length} chapters loaded`);
});

manager.on('ChapterStarted', (player, track, payload) => {
  console.log(`Chapter: ${payload.chapter.name}`);
});
```

## LavaLyrics <Badge text="lavalyrics-plugin" variant="caution" />

Lyrics API hub — aggregates from LavaSrc, java-timed-lyrics, and other sources.

### Methods

```typescript
// Search for lyrics globally across all providers
const searchResults = await player.node.lyrics.search('never gonna give you up');
// [{ videoId: '...', title: '...' }]

// Get lyrics by videoId
const lyricsResult = await player.node.lyrics.getLyrics(searchResults[0].videoId);

// Subscribe to real-time lyrics lines during playback
await player.node.lyrics.subscribe(player.guildId);
await player.node.lyrics.unsubscribe(player.guildId);

// Convenience methods on the player
const lyricsData = await player.getCurrentLyrics(); // Wrapper for node.lyrics.getCurrent
const lyricsByTrack = await player.getLyrics(track); // Wrapper for node.lyrics.get
player.subscribeLyrics();
player.unsubscribeLyrics();
```

### Events

```typescript
manager.on('LyricsFound', (player, track, payload) => {
  console.log(`Lyrics found via ${payload.lyrics.sourceName}`);
});

manager.on('LyricsLine', (player, track, payload) => {
  console.log(`[${payload.lineIndex}] ${payload.line.line}`);
});

manager.on('LyricsNotFound', (player, track, payload) => {
  console.log('No lyrics available');
});
```

## java-timed-lyrics / lyrics.kt <Badge text="java-lyrics-plugin / lyrics" variant="caution" />

These plugins expose a different API from LavaLyrics. Use `node.timedLyrics`:

```typescript
// Get lyrics for the current player
const result = await player.node.timedLyrics.getCurrent(player.guildId);

// Get by YouTube video ID
const result2 = await player.node.timedLyrics.getByVideoId('dQw4w9WgXcQ');

// Search for lyrics
const results = await player.node.timedLyrics.search('never gonna give you up');
const genius = await player.node.timedLyrics.search('never gonna give you up', 'genius');
```

## LavaDSPX Filters <Badge text="LavaDSPX-Plugin" variant="caution" />

Additional audio filters: high-pass, low-pass, normalization, echo.

```typescript
// Low-pass filter (cuts high frequencies)
await player.filterManager.dspxPlugin.toggleLowPass(1.0, 80);

// High-pass filter (cuts low frequencies)
await player.filterManager.dspxPlugin.toggleHighPass(1.0, 80);

// Normalization (prevents clipping)
await player.filterManager.dspxPlugin.toggleNormalization(0.75, true);

// Echo effect
await player.filterManager.dspxPlugin.toggleEcho(0.5, 0.5);
```

## LavaSearch <Badge text="lavasearch-plugin" variant="caution" />

Advanced search returning albums, artists, playlists, and text results.

```typescript
// Use audioSearch for rich results
const results = await player.audioSearch({
  query: 'daft punk',
  source: 'spsearch',
  types: ['track', 'album', 'artist', 'playlist']
});

console.log(results.tracks);   // Track[]
console.log(results.albums);   // Album[]
console.log(results.artists);  // Artist[]
console.log(results.playlists); // Playlist[]
```

## NodeLink Gapless Playback <Badge text="NodeLink only" variant="tip" />

Pre-load the next track for zero-silence transitions.

```typescript
// Pre-load next track while current is playing
await player.node.setNextTrackGapLess(player);

// Or specify a track explicitly
await player.node.setNextTrackGapLess(player, nextTrack);

// Remove the preloaded track
await player.node.removeNextTrackGapLess(player);
```

## NodeLink Audio Mixer <Badge text="NodeLink only" variant="tip" />

Overlay TTS, SFX, or background music on top of the main track.

```typescript
// Add a mix layer (volume is 0.0–1.0)
const layer = await player.node.addMixerLayer(player, ttsTrack, 0.9);
console.log(layer.id); // mixId

// List active layers
const layers = await player.node.listMixerLayers(player);

// Update volume of a layer
await player.node.updateMixerLayerVolume(player, layer.id, 0.5);

// Remove a layer
await player.node.removeMixerLayer(player, layer.id);
```

### Mixer Events

```typescript
manager.nodeManager.on('nodeLinkEvent', (node, eventType, player, track, payload) => {
  if (eventType === 'MixStartedEvent') {
    console.log(`Mix started: ${payload.mixId} — ${payload.track.info.title}`);
  }
  if (eventType === 'MixEndedEvent') {
    console.log(`Mix ended: ${payload.mixId} — reason: ${payload.reason}`);
  }
});
```
