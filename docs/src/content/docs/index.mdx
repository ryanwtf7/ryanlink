## Why Ryanlink?

<CardGrid stagger>
  <Card title="Full Lavalink v4 Support" icon="rocket">
    Complete implementation of the Lavalink v4 REST and WebSocket API — players, filters, sessions, route planner, and all plugin endpoints.
  </Card>
  <Card title="50+ Search Sources" icon="magnifier">
    YouTube, Spotify, Apple Music, Deezer, Tidal, SoundCloud, JioSaavn, Yandex Music, VK Music, Qobuz, Pandora, and many more via LavaSrc and LavaSearch.
  </Card>
  <Card title="Intelligent Autoplay" icon="random">
    Multi-source fetch chain with weighted scoring, circular history buffer, pre-fetch, and keyword exclusion — all built-in.
  </Card>
  <Card title="NodeLink Ready" icon="star">
    Full NodeLink support: Audio Mixer, Fading, Gapless preloading, Direct streaming, Eternal Box, YouTube config, and cluster workers.
  </Card>
</CardGrid>
