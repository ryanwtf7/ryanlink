import { Badge } from '@astrojs/starlight/components'

###  <Badge text="2.0" variant="tip" />
- **Zero-Config Persistence** <Badge text="Reliable" variant="success" />
  - Automated state recovery for queues and playback positions across restarts.
  - Built-in drivers for **Redis**, **Local Disk**, and **In-Memory** storage.
- **Atomic Migration** <Badge text="Seamless" variant="note" />
  - Automated node failover with zero-interruption playback during disconnections.
  - Weighted load balancing scoring (CPU, Memory, Frame Deficit).

### Performance & Memory
- **Memory-Optimized TrackRegistry** <Badge text="Efficient" variant="caution" />
  - High-performance reference mapping for queues with **10,000+ tracks**.
  - Minimized RAM footprint by deferring track metadata resolution.
- **Fisher-Yates Shuffle** <Badge text="Fair" variant="note" />
  - Statistically unbiased randomization for high-fidelity audio shuffling.

### Extensive Sources
- **Unified Connectivity** <Badge text="20+ Platforms" variant="tip" />
  - Native support for **Spotify**, **Apple Music**, **Deezer**, **Tidal**, **JioSaavn**, and more.
  - Seamless integration for **YouTube**, **SoundCloud**, **TikTok**, **Instagram**, and **Amazon Music**.
  - Optimized for **Regional Platforms** including **VK Music**, **Gaana**, and **Audiomack**.
  - Built-in support for **TTS Engine** (**Flowery**, **Google Cloud**) and **Local/HTTP** files.

### Audio Intelligence
- **High-Fidelity Filters** <Badge text="Professional" variant="tip" />
  - Non-destructive filter stacking with atomic updates.
  - Professional presets including **8D**, **Lofi**, **Radio**, **Nightcore**, and **Vaporwave**.
- **Automated Self-Healing** <Badge text="Robust" variant="danger" />
  - Integrated track resolve retries (3 attempts) before auto-purging flaky sources.
  - Lifecycle hooks for `onTrackStart`, `onQueueEnd`, and `onNodeFailover`.

### Plugin Ecosystem
- **SponsorBlock** <Badge text="Integrated" variant="tip" />
  - Automatic detection and skipping of sponsors, intros, and music-only segments.
- **LavaLyrics** <Badge text="Real-time" variant="note" />
  - Time-synced lyrics support with specialized event tracking for immersive UI.
- **NodeLink Gapless** <Badge text="Perfect" variant="success" />
  - Zero-silence playback between consecutive tracks for a seamless listening experience.
