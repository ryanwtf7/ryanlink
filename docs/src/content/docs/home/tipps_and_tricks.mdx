### Skip Broken-Tracks

Some tracks are broken by not containing duration or broken duration. Due to the nature of Flexibility ryanlink does not handle those scenarios, so you need to check it yourself.
There are utils for that built in, and this is how you could do it:

```ts
const { tracks } = await player.search('Adele')

// short hand:
const validTracks = tracks.filter(client.ryanlink.utils.isNotBrokenTrack)
// or with options:
const validTracks = tracks.filter((t) => client.ryanlink.utils.isNotBrokenTrack(t, 29e3))

// then you can add it to the queue.
await player.queue.add(validTracks)
```
