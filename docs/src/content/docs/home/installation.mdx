## Requirements

- **Node.js** v18+ (v20+ recommended) or **Bun** v1.0+
- A running **Lavalink v4** server

## Install

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install ryanlink
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash
    bun add ryanlink
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add ryanlink
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add ryanlink
    ```
  </TabItem>
</Tabs>

## TypeScript

Ryanlink ships with full TypeScript types. No `@types` package needed.

```typescript
import { RyanlinkManager, RyanlinkNode, Player, Track } from 'ryanlink';
```
